# Multivendor Network Engineering Lab Portfolio

## 📌 Overview
This project showcases hands-on networking labs focused on multivendor routing, switching, and multicast technologies using Cisco, Juniper, Nokia, and Linux environments.

## 🔧 Technologies & Protocols
- VLANs & Inter-VLAN Routing  
- Spanning Tree Protocol (STP)  
- OSPF, BGP, IS-IS, RIP  
- Static Routing  
- Multicast Routing (PIM)  
- Containerlab (network emulation)  

## 🧠 Skills Demonstrated
- Network configuration and troubleshooting  
- Multi-vendor environment management  
- Routing protocol implementation and validation  
- Network design and analysis
